## 12. Disaster Recovery (DR)

### Definition

Disaster Recovery is the process of restoring systems, applications, and data after a disaster.

### Examples of Disasters

* Fire
* Flood
* Server Failure
* Cyber Attack
* Human Error

### Goals

* Minimize Downtime
* Prevent Data Loss
* Ensure Business Continuity

### Important Terms

#### RPO (Recovery Point Objective)

Maximum acceptable data loss.

#### RTO (Recovery Time Objective)

Maximum acceptable recovery time.

### DR Techniques

* Backup
* Replication
* Failover
* Recovery Planning

---

## 13. Backup and Disaster Recovery in Cloud

### Backup

Creating copies of data.

### Disaster Recovery

Restoring complete systems and services after failure.

### Benefits

* Data Protection
* Fast Recovery
* Reduced Downtime
* Business Continuity

## Technologies in Next-Gen Cloud Computing

### 1. Unikernels

A lightweight operating system designed for a single application.

#### Advantages

* Fast Boot Time
* Small Size
* Better Security
* High Performance
* Less Resource Usage

---

### 2. Blockchain

A distributed digital ledger technology that stores data in linked blocks across multiple computers.

#### Features

* Decentralization
* Transparency
* Security
* No Single Point of Failure

#### Example

Bitcoin (2009)

---

### 3. Container-as-a-Service (CaaS)

A cloud service model used to deploy and manage containers.

#### Examples

* Docker
* Kubernetes
* AWS ECS
* AWS EKS

#### Benefits

* Faster Deployment
* Scalability
* Portability
* Resource Efficiency

---

### 4. Software Defined Networking (SDN)

A networking architecture that allows centralized control of networks using software.

#### Advantages

* Centralized Management
* Automation
* Scalability
* Better Security
* Reduced Cost
