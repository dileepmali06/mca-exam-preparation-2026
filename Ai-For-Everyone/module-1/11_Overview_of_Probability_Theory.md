# Introduction

Real-world problems are uncertain. Artificial Intelligence cannot always
make decisions using fixed rules because information is often
incomplete.

Probability Theory provides a mathematical framework for measuring
uncertainty and making intelligent predictions.

------------------------------------------------------------------------

# Why Probability is Needed in AI

AI systems frequently work with uncertain information.

Examples:

-   Weather prediction
-   Medical diagnosis
-   Fraud detection
-   Spam email filtering
-   Self-driving cars

Probability helps AI estimate the likelihood of different outcomes
before making decisions.

------------------------------------------------------------------------

# Definition

> **Probability is the numerical measure of the likelihood that an event
> will occur.**

Probability values always lie between **0 and 1**.

-   0 = Impossible Event
-   1 = Certain Event

------------------------------------------------------------------------

# Basic Concepts

## 1. Random Experiment

A Random Experiment is an experiment whose outcome cannot be predicted
with certainty.

Examples:

-   Tossing a coin
-   Rolling a dice
-   Drawing a card

------------------------------------------------------------------------

## 2. Sample Space

The set of all possible outcomes of a random experiment.

Examples:

Coin Toss

``` text
S = {Head, Tail}
```

Dice Roll

``` text
S = {1,2,3,4,5,6}
```

------------------------------------------------------------------------

## 3. Event

An Event is a specific outcome or a group of outcomes from the sample
space.

Example:

``` text
Even Numbers = {2,4,6}
```

------------------------------------------------------------------------

# Probability Formula

\[
P(E)=`\frac{\text{Number of Favorable Outcomes}}{\text{Total Number of Outcomes}}`{=tex}
\]

Example:

Probability of getting Head:

``` text
Favorable Outcomes = 1
Total Outcomes = 2

P(Head)=1/2=0.5
```

# MCQs

### Q1. Probability measures:

A. Speed

B. Chance of an event

C. Memory

D. Storage

**Answer:** B. Chance of an event

**Explanation:** Probability measures the likelihood of an event.

------------------------------------------------------------------------

### Q2. Probability always lies between:

A. 0 and 1

B. 1 and 10

C. -1 and 1

D. 0 and 100

**Answer:** A. 0 and 1

**Explanation:** Probability ranges from impossible (0) to certain (1).

------------------------------------------------------------------------

### Q3. Which is a Random Experiment?

A. Saving a file

B. Tossing a coin

C. Printing a page

D. Installing software

**Answer:** B. Tossing a coin

**Explanation:** Its outcome cannot be predicted with certainty.

------------------------------------------------------------------------

### Q4. Sample Space is:

A. One outcome

B. All possible outcomes

**Answer:** B. All possible outcomes

### Q5. An Event is:

**Answer:** B. Desired outcome
