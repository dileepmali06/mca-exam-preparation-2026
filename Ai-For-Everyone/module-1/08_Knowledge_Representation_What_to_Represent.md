# Introduction

Artificial Intelligence cannot make intelligent decisions unless it
knows **what kind of knowledge should be stored**.

Knowledge Representation is not limited to storing facts. An AI system
must represent different kinds of information about the real world so
that it can reason, learn, and solve problems.

------------------------------------------------------------------------

# What Should Be Represented?

An intelligent AI system should represent:

1.  Objects (Subjects)
2.  Events
3.  Performance Knowledge
4.  Meta-Knowledge
5.  Facts
6.  Knowledge Base

------------------------------------------------------------------------

# 1. Objects (Subjects)

## Definition

Objects are real-world entities about which an AI system stores
knowledge.

### Examples

-   Person
-   Car
-   Dog
-   Hospital
-   Mobile Phone

Example:

``` text
Car
│
├── Color
├── Speed
├── Brand
└── Engine
```

Applications:

-   Self-driving cars
-   Face recognition
-   Object detection
-   Smart assistants

------------------------------------------------------------------------

# 2. Events

## Definition

Events are activities or occurrences that happen at a particular time.

### Examples

-   Rainfall
-   Login
-   Payment
-   Accident
-   Meeting

Example:

``` text
Meeting
    │
Reminder
    │
Notification
```

Applications:

-   Google Calendar
-   Banking
-   Security systems
-   Smart homes

------------------------------------------------------------------------

# 3. Performance Knowledge

## Definition

Performance Knowledge refers to the knowledge required to perform a
particular task correctly.

### Example

Making Tea

``` text
Boil Water
     │
Add Tea
     │
Add Milk
     │
Serve
```

Applications:

-   Industrial Robots
-   Warehouse Automation
-   Manufacturing
-   Medical Robots

------------------------------------------------------------------------

# 4. Meta-Knowledge

## Definition

Meta-Knowledge is knowledge about existing knowledge.

It describes:

-   Source of knowledge
-   Reliability
-   Confidence
-   Scope

Example:

``` text
Disease Prediction

Confidence = 95%
```

Applications:

-   Medical Diagnosis
-   Expert Systems
-   AI Assistants

------------------------------------------------------------------------

# 5. Facts

## Definition

Facts are verified and true statements stored in an AI system.

Examples:

-   Delhi is the capital of India.
-   Water boils at 100°C.
-   Earth revolves around the Sun.

Applications:

-   Search Engines
-   Educational Systems
-   Knowledge Graphs

------------------------------------------------------------------------

# 6. Knowledge Base

## Definition

A Knowledge Base is a structured repository containing facts, rules,
concepts, and relationships used by AI systems.

Example:

``` text
Knowledge Base
│
├── Facts
├── Rules
├── Concepts
└── Relationships
```

Applications:

-   Chatbots
-   IBM Watson
-   Customer Support Systems
-   Medical AI

------------------------------------------------------------------------

# Overall Structure

``` text
Knowledge Representation
        │
        ├── Objects
        ├── Events
        ├── Performance
        ├── Meta-Knowledge
        ├── Facts
        └── Knowledge Base
```

# Important Definitions

**Objects:** Real-world entities represented by AI.

**Events:** Activities that occur over time.

**Performance Knowledge:** Knowledge required to perform tasks.

**Meta-Knowledge:** Knowledge about existing knowledge.

**Facts:** Verified information.

**Knowledge Base:** Repository containing structured knowledge.

# MCQs

### Q1. Which of the following represents a real-world entity?

A. Event

B. Object

C. Fact

D. Knowledge Base

**Answer:** B. Object

**Explanation:** Objects represent real-world entities like people,
cars, or buildings.

------------------------------------------------------------------------

### Q2. Which component stores facts and rules?

A. Performance

B. Event

C. Knowledge Base

D. Object

**Answer:** C. Knowledge Base

**Explanation:** A Knowledge Base stores structured facts and rules.

------------------------------------------------------------------------

### Q3. Meta-Knowledge means:

A. Knowledge about knowledge

B. Hardware knowledge

C. Programming knowledge

D. Database knowledge

**Answer:** A. Knowledge about knowledge

**Explanation:** It describes information about existing knowledge.

------------------------------------------------------------------------

### Q4. Which of the following is an event?

A. Hospital

B. Rainfall

C. Car

D. Laptop

**Answer:** B. Rainfall

**Explanation:** Rainfall is an event that occurs over time.

------------------------------------------------------------------------

### Q5. Facts are:

A. Assumptions

B. Verified information

C. Images

D. Programs

**Answer:** B. Verified information

**Explanation:** Facts are true statements stored in AI systems.

------------------------------------------------------------------------

### Q6. Performance Knowledge helps AI to:

A. Increase storage

B. Perform tasks

C. Compress files

D. Draw diagrams

**Answer:** B. Perform tasks

**Explanation:** It guides AI in completing tasks correctly.

------------------------------------------------------------------------

### Q7. Which is NOT a component of "What to Represent"?

A. Objects

B. Compiler

C. Facts

D. Events

**Answer:** B. Compiler

**Explanation:** A compiler is unrelated to Knowledge Representation.

------------------------------------------------------------------------

### Q8. AI uses Knowledge Representation for:

A. Reasoning

B. Decision Making

C. Problem Solving

D. All of the Above

**Answer:** D. All of the Above

**Explanation:** KR supports intelligent reasoning and decision making.

------------------------------------------------------------------------

### Q9. A chatbot mainly uses:

A. Knowledge Base

B. Printer

C. Scanner

D. Monitor

**Answer:** A. Knowledge Base

**Explanation:** Chatbots retrieve answers from a structured knowledge
base.

------------------------------------------------------------------------

### Q10. Which statement is correct?

A. AI only stores facts.

B. AI only stores objects.

C. AI represents multiple types of knowledge.

D. AI does not need knowledge.

**Answer:** C. AI represents multiple types of knowledge.

**Explanation:** AI requires different kinds of knowledge for
intelligent behavior.

An intelligent AI system represents objects, events, performance
