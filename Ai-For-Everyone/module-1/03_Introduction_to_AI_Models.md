# 1. Introduction

Artificial Intelligence systems perform intelligent tasks using AI
Models.

An AI Model is a trained computational model that learns patterns from
data and uses those patterns to make predictions, decisions,
recommendations, or generate outputs.

Without AI Models, modern AI applications such as ChatGPT, Google Maps,
Netflix Recommendations, and Self-Driving Cars would not exist.

------------------------------------------------------------------------

# 2. What is an AI Model?

## Definition

> **An AI Model is a trained computational model that learns patterns
> from data and uses those learned patterns to make predictions,
> decisions, classifications, recommendations, or generate outputs.**

------------------------------------------------------------------------

# 3. Why Do We Need AI Models?

Traditional software follows fixed rules written by programmers.

However, many real-world problems cannot be solved using fixed rules.

Examples:

-   Face Recognition
-   Language Translation
-   Recommendation Systems
-   Spam Detection
-   Fraud Detection

Instead of writing thousands of rules, AI Models learn directly from
data.

------------------------------------------------------------------------

# 4. AI Model vs Algorithm

## Algorithm

An algorithm is a step-by-step procedure or set of instructions used to
solve a problem.

Example:

    Input
    ↓

    Process

    ↓

    Output

Algorithms do not learn automatically.

------------------------------------------------------------------------

## AI Model

An AI Model is created after training an algorithm using large amounts
of data.

Formula:

    Algorithm
          +
    Training Data
          +
    Learning
          =
    AI Model

An AI Model can make intelligent predictions on new data.

------------------------------------------------------------------------

# 5. Types of AI Models

## Rule-Based Models

-   Work using predefined rules.
-   Do not learn automatically.
-   Require manual programming.

Example:

-   Calculator
-   Rule-based Expert Systems

------------------------------------------------------------------------

## Machine Learning Models

-   Learn from data.
-   Improve performance through experience.
-   Can make predictions on unseen data.

Examples:

-   Spam Detection
-   Recommendation Systems

------------------------------------------------------------------------

## Self-Learning Models

These models continuously improve using new data and experience.

Example:

-   Google Maps
-   Online Recommendation Systems

------------------------------------------------------------------------

## Pre-Programmed Models

These systems execute predefined instructions without learning.

Example:

-   Basic Automation Software

------------------------------------------------------------------------
