# Why Knowledge Representation Exists

Artificial Intelligence cannot think like humans unless knowledge is
stored in a structured form.

Humans naturally understand facts, relationships, and experiences, but
computers require knowledge to be represented in a
machine-understandable format.

Knowledge Representation provides the bridge between real-world
knowledge and intelligent computer systems.

------------------------------------------------------------------------

# Introduction

Knowledge Representation (KR) is one of the core components of
Artificial Intelligence.

It provides a framework to represent, organize, and manipulate knowledge
so that AI systems can reason, solve problems, and make intelligent
decisions.

------------------------------------------------------------------------

# Definition

> **Knowledge Representation is the process of representing real-world
> knowledge in a structured format so that computers can understand,
> reason, and make intelligent decisions.**

------------------------------------------------------------------------

# Book Perspective

According to the syllabus, an intelligent system requires explicit
knowledge representation for reasoning and decision making. Knowledge
Representation helps organize information, solve complex problems, and
learn from data.

------------------------------------------------------------------------

# Data vs Information vs Knowledge

  ------------------------------------------------------------------------
  Data           Information                    Knowledge
  -------------- ------------------------------ --------------------------
  Raw facts      Processed data                 Information with
                                                understanding and
                                                experience

  ------------------------------------------------------------------------

Example:

``` text
Data:
45°C

↓

Information:
Today's temperature is 45°C.

↓

Knowledge:
It is extremely hot. Avoid going outside during peak afternoon hours.
```

------------------------------------------------------------------------

# Knowledge Representation and Reasoning (KRR)

Knowledge Representation stores knowledge.

Reasoning uses stored knowledge to derive new conclusions.

Example:

``` text
Knowledge:

All humans breathe.

Dilip is a human.

↓

Reasoning

↓

Therefore,

Dilip breathes.
```

------------------------------------------------------------------------

# Knowledge Representation Workflow

``` text
Real World
     │
Collect Knowledge
     │
Represent Knowledge
     │
Store in Knowledge Base
     │
Reasoning
     │
Decision
```

A Knowledge Base is a structured collection of facts, rules, and
information that AI systems use for reasoning and intelligent decision
making.
