# Introduction

Artificial Intelligence (AI) is a broad field that aims to build
intelligent machines capable of performing tasks that normally require
human intelligence.

Machine Learning (ML) is a subset of AI that allows computers to learn
from data instead of being explicitly programmed for every task.

------------------------------------------------------------------------

# Brief History of AI

## 1950

-   Alan Turing proposed the idea of machine intelligence and introduced
    the Turing Test.

## 1956

-   John McCarthy coined the term **Artificial Intelligence** at the
    Dartmouth Conference.

## 1960s--1980s

-   Expert Systems became popular.
-   AI relied mainly on manually written rules.

## 1990s

-   Machine Learning gained importance because systems could learn from
    data.

## 2010s--Present

-   Growth of Big Data, GPUs, and Deep Learning.
-   Rise of ChatGPT, autonomous vehicles, recommendation systems, and
    Generative AI.

------------------------------------------------------------------------

# Artificial Intelligence (AI)

## Definition

> **Artificial Intelligence is the branch of computer science that
> enables machines to perform tasks that normally require human
> intelligence.**

Examples:

-   Chatbots
-   Face Recognition
-   Self-driving Cars
-   Virtual Assistants
-   Language Translation

------------------------------------------------------------------------

# Machine Learning (ML)

## Definition

> **Machine Learning is a subset of Artificial Intelligence that enables
> computers to learn from data and improve their performance without
> being explicitly programmed.**

Instead of writing every rule manually, ML algorithms identify patterns
from data and make predictions.

------------------------------------------------------------------------

# Relationship between AI and ML

``` text
Artificial Intelligence
        │
        ├── Machine Learning
        │       ├── Deep Learning
        │       ├── Supervised Learning
        │       ├── Unsupervised Learning
        │       └── Reinforcement Learning
        │
        ├── Robotics
        ├── NLP
        └── Computer Vision
```

------------------------------------------------------------------------

# Traditional Programming vs Machine Learning

## Traditional Programming

``` text
Data + Rules
      │
    Program
      │
    Output
```

## Machine Learning

``` text
Data + Correct Answers
          │
Learning Algorithm
          │
        Model
          │
     New Prediction
```

------------------------------------------------------------------------

# AI vs Machine Learning

  Artificial Intelligence           Machine Learning
  --------------------------------- ------------------------------------
  Broad field                       Subset of AI
  Simulates human intelligence      Learns from data
  May use rules or learning         Learns patterns from examples
  Focuses on intelligent behavior   Focuses on prediction and learning

# MCQs

### Q1. Machine Learning is a:

A. Programming Language

B. Database

C. Subset of Artificial Intelligence

D. Operating System

**Answer:** C. Subset of Artificial Intelligence

**Explanation:** Machine Learning is one of the major branches of AI.

------------------------------------------------------------------------

### Q2. Artificial Intelligence aims to:

A. Compress files

B. Mimic human intelligence

C. Increase RAM

D. Build websites only

**Answer:** B. Mimic human intelligence

**Explanation:** AI enables machines to perform intelligent tasks.

------------------------------------------------------------------------

### Q3. Machine Learning learns from:

A. Data

B. Keyboard

**Explanation:** ML identifies patterns from data.

# One-Line Summary

Artificial Intelligence is the broad field of building intelligent
machines, while Machine Learning is a subset of AI that enables systems
to learn from data and improve automatically.
