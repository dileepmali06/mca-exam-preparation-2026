# Introduction

Machine Learning (ML) enables computers to learn patterns from data
instead of following manually written rules.

The system learns from examples, builds a model, and uses that model to
make predictions on new data.

------------------------------------------------------------------------

# Machine Learning Workflow

``` text
Collect Data
     │
Clean Data
     │
Feature Selection
     │
Model Training
     │
Model Testing
     │
Prediction
```

------------------------------------------------------------------------

# Important Concepts

## Data

Data is the information used to train a machine learning model.

Examples:

-   Student marks
-   House prices
-   Customer purchases

------------------------------------------------------------------------

## Features

Features are input variables used by the model.

Examples:

-   House Size
-   Bedrooms
-   Location

------------------------------------------------------------------------

## Label (Target)

The expected output that the model learns to predict.

Example:

-   House Price

------------------------------------------------------------------------

## Model

A Machine Learning Model is a trained mathematical representation that
learns patterns from data and makes predictions.

------------------------------------------------------------------------

## Training

Training is the process of teaching the model using data.

------------------------------------------------------------------------

## Testing

Testing evaluates the trained model using unseen data.

------------------------------------------------------------------------

## Prediction

Prediction is the output generated by the trained model for new input
data.

------------------------------------------------------------------------

# Machine Learning Pipeline

``` text
Training Data
      │
Learning Algorithm
      │
Trained Model
      │
Testing Data
      │
Prediction
```

------------------------------------------------------------------------

# Types of Machine Learning

## 1. Supervised Learning

Uses labeled data.

Examples:

-   Spam Detection
-   House Price Prediction
-   Disease Prediction

------------------------------------------------------------------------

## 2. Unsupervised Learning

Uses unlabeled data to discover hidden patterns.

Examples:

-   Customer Segmentation
-   Data Clustering
-   Market Basket Analysis

------------------------------------------------------------------------

## 3. Reinforcement Learning

Learns through rewards and penalties.

Examples:

-   Robotics
-   Self-Driving Cars
-   Game Playing

------------------------------------------------------------------------

# Real-World Applications

-   Google Photos Face Recognition
-   Fraud Detection

# MCQs

### Q1. Machine Learning learns from:

A. CPU

B. Data

C. Printer

D. Keyboard

**Answer:** B. Data

**Explanation:** Data is the primary source of learning.

------------------------------------------------------------------------

### Q2. Features are:

A. Output values

B. Input variables

C. Errors

D. Predictions

**Answer:** B. Input variables

**Explanation:** Features are inputs to the ML model.

------------------------------------------------------------------------

### Q3. Label refers to:

A. Input

B. Target output

C. Feature

D. Dataset

**Answer:** B. Target output

**Explanation:** Labels are the expected outputs.

------------------------------------------------------------------------

### Q4. Training is:

A. Evaluating a model

B. Teaching a model using data

C. Deleting data

D. Compressing files

**Answer:** B. Teaching a model using data

**Explanation:** Training helps the model learn patterns.

------------------------------------------------------------------------

### Q5. Testing is used to:

A. Train a model

B. Evaluate a trained model

C. Create data

D. Delete a model

**Answer:** B. Evaluate a trained model

**Explanation:** Testing measures model performance.

------------------------------------------------------------------------

### Q6. Supervised Learning uses:

A. Unlabeled data

B. Labeled data

C. Random data only

D. Images only

**Answer:** B. Labeled data

**Explanation:** Correct outputs are provided during training.

------------------------------------------------------------------------

### Q7. Unsupervised Learning mainly finds:

A. Hidden patterns

B. Website bugs

C. CPU errors

D. File sizes

**Answer:** A. Hidden patterns

**Explanation:** It discovers structure without labels.

------------------------------------------------------------------------

### Q8. Reinforcement Learning learns through:

A. Labels

B. Reward and penalty

C. Sorting

D. Compression

**Answer:** B. Reward and penalty

**Explanation:** Agents improve using feedback.

------------------------------------------------------------------------

### Q9. Which company uses ML for movie recommendations?

A. Netflix

B. VLC

C. Notepad

D. Paint

**Answer:** A. Netflix

**Explanation:** Netflix uses ML-based recommendation systems.

------------------------------------------------------------------------

### Q10. The output of a trained model is called:

A. Feature

B. Dataset

C. Prediction

D. Label

**Answer:** C. Prediction

**Explanation:** Predictions are generated for new data.
