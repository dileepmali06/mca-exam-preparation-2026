# Why Do We Need Backpropagation?

A neural network first makes a prediction using Forward Propagation.

If the prediction is wrong, the network must know:

-   Which weight caused the error?
-   How much should that weight change?
-   In which direction should it change?

Backpropagation solves this problem.

------------------------------------------------------------------------

# What is Backpropagation?

Backpropagation (Backward Propagation of Errors) is a learning algorithm
that propagates errors from the output layer back to the input layer and
updates the network weights to reduce prediction error.

------------------------------------------------------------------------

# Why is it Called Backpropagation?

``` text
Input
  │
  ▼
Hidden Layer
  │
  ▼
Output

Backpropagation

Output
  ▲

  ▲

```

Data moves forward, but error moves backward.

# Types of Backpropagation

## 1. Static Backpropagation

Characteristics:

-   Static input
-   Static output
-   Used for classification problems
-   Example: OCR, Image Classification

------------------------------------------------------------------------

## 2. Recurrent Backpropagation

Characteristics:

-   Sequential data
-   Feedback mechanism
-   Used for sequence learning
-   Example: Speech Recognition, Chatbots

------------------------------------------------------------------------

# Static vs Recurrent Backpropagation

  Static Backpropagation   Recurrent Backpropagation
  ------------------------ ---------------------------
  Fixed input/output       Sequential input/output
  Image Classification     Speech & Language
  OCR                      Chatbots

------------------------------------------------------------------------

# Objective of Backpropagation

-   Minimize the loss function.
-   Improve prediction accuracy.
-   Update weights efficiently.
-   Train neural networks.

------------------------------------------------------------------------

# Working Steps

1.  Receive input.
2.  Perform Forward Propagation.
3.  Generate prediction.
4.  Compare prediction with actual output.
5.  Calculate loss.
6.  Propagate error backward.
7.  Compute gradients.
8.  Update weights.
9.  Repeat until loss becomes minimum.

# One-Line Definitions

-   **Backpropagation:** Algorithm that updates weights using prediction
    errors.
-   **Gradient:** Direction of maximum change in loss.
-   **Weight Update:** Adjusting parameters to improve predictions.
-   **Loss Function:** Measures prediction error.

# Practice MCQs

## Q1. Backpropagation is also known as:

A. Forward Learning

B. Backward Propagation of Errors

C. Gradient Mapping

D. Weight Initialization

**Answer:** B

**Explanation:** It propagates errors from output to input.

------------------------------------------------------------------------

## Q2. Main objective of Backpropagation is:

A. Increase neurons

B. Minimize loss

C. Increase epochs

D. Increase batch size

**Answer:** B

**Explanation:** It updates weights to reduce prediction error.

------------------------------------------------------------------------

## Q3. Error moves from:

A. Input to Output

B. Hidden to Output

C. Output to Input

D. Randomly

**Answer:** C

**Explanation:** Errors travel backward through the network.

------------------------------------------------------------------------

## Q4. Static Backpropagation is mainly used for:

A. OCR

B. Translation

C. Speech

D. Forecasting

**Answer:** A

**Explanation:** It works with fixed input-output problems.

------------------------------------------------------------------------

## Q5. Recurrent Backpropagation is suitable for:

A. Images

B. Sequential Data

C. Tables

D. Regression only

**Answer:** B

**Explanation:** It is designed for sequence-based learning.

------------------------------------------------------------------------

## Q6. Which optimizer commonly works with Backpropagation?

D. KNN

# Final Summary

Backpropagation is one of the most important algorithms in Deep
Learning. It sends the prediction error backward through the network,
computes gradients, and updates weights so that the loss decreases and
the model becomes more accurate after every training iteration.
