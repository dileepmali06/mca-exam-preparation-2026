# Introduction

Artificial Intelligence (AI) is a technology that enables machines to perform tasks that normally require human intelligence, such as learning, reasoning, decision-making, and problem-solving.

Machine Learning (ML) is a subset of AI that allows computers to learn patterns from data without being explicitly programmed.

Deep Learning (DL) is a subset of Machine Learning that uses Artificial Neural Networks (ANNs) with multiple hidden layers to automatically learn complex patterns from large datasets.

# Artificial Intelligence (AI)

## Definition

**Artificial Intelligence (AI)** is the ability of machines to simulate human intelligence and perform tasks such as learning, reasoning, problem-solving, and decision-making.

### Examples

- ChatGPT
- Siri
- Alexa
- Self-driving cars
- Face Recognition Systems

---

# Machine Learning (ML)

## Definition

**Machine Learning** is a branch of Artificial Intelligence that enables computers to learn from data without being explicitly programmed.

Instead of writing rules manually, the computer discovers patterns from data.

### Examples

- Spam Email Detection
- Movie Recommendation
- Credit Card Fraud Detection

---

# Deep Learning (DL)

## Definition

**Deep Learning** is a subset of Machine Learning that uses Artificial Neural Networks with multiple hidden layers to learn complex representations of data automatically.

Unlike traditional Machine Learning, Deep Learning performs automatic feature extraction.

---

# Why Do We Need Deep Learning?

Traditional Machine Learning struggles when:

- Dataset is very large.
- Data is complex.
- Images are involved.
- Speech is involved.
- Natural language is involved.

Deep Learning solves these problems by learning features automatically.

---

# Artificial Neural Network (ANN)

## Definition

**Artificial Neural Network (ANN)** is a computational model inspired by the human brain that consists of interconnected neurons used to learn patterns from data.

ANN is the foundation of Deep Learning.

---

| Neurons | Artificial Neurons |
| Signals | Inputs |

# Optimization

## Definition

**Optimization** is the process of finding the best values of model parameters that minimize the loss function and improve prediction accuracy.

### Goal of Optimization

- Reduce Error
- Improve Accuracy
- Train the Model Efficiently

---

# Loss Function

## Definition

**Loss Function** measures the difference between the predicted output and the actual output.

The objective during training is to minimize the loss.

---
