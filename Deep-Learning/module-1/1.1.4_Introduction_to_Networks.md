# Why Do We Need Neural Networks?

Traditional programming cannot easily solve complex tasks like image
recognition, speech recognition, recommendation systems and language
understanding.

Neural Networks automatically learn patterns from data.

------------------------------------------------------------------------

# What is a Neural Network?

A Neural Network is a collection of interconnected artificial neurons
inspired by the human brain. It learns hidden relationships from data
and makes predictions.

------------------------------------------------------------------------

# Architecture

``` text
Input Layer
      │
      ▼
Hidden Layer 1
      │
      ▼
Hidden Layer 2
      │
      ▼
Output Layer
```

------------------------------------------------------------------------

# Input Layer

-   Receives data.
-   Passes information to the hidden layers.
-   Does not perform learning.

Examples: - Images - Text - Audio - Numerical data

------------------------------------------------------------------------

# Hidden Layer

-   Learns patterns.
-   Performs mathematical computations.
-   Uses weights, biases and activation functions.

Example:

Image ↓ Edges ↓ Eyes ↓ Face

------------------------------------------------------------------------

# Output Layer

Produces the final prediction.

Example:

Cat = 98% Dog = 2%

Prediction = Cat

------------------------------------------------------------------------

# Feed Forward Process

``` text
Input
 ↓
Input Layer
 ↓
Hidden Layer
 ↓
Output Layer
 ↓
Prediction
```

------------------------------------------------------------------------

# Feed Forward Neural Network

Characteristics:

-   Simplest Neural Network.
-   Data moves only from input to output.
-   No loops.
-   Used for regression and classification.

------------------------------------------------------------------------

# Types of Neural Networks

## Feed Forward Neural Network (FFNN)

Uses: - Classification - Regression

## Convolutional Neural Network (CNN)

Uses: - Image Recognition - Face Detection - Medical Imaging

## Recurrent Neural Network (RNN)

Uses: - Language Processing - Speech Recognition - Time-Series Data

-   Learns automatically.

# Limitations
