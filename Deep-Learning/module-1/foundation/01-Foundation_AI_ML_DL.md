# Why Do We Need AI?

Traditional programming requires writing rules for every possible
situation.

Example:

-   If object has 4 legs
-   If it has a tail
-   If it barks

Then classify it as a dog.

This approach becomes impossible for real-world problems.

Researchers asked:

> Can a computer learn patterns instead of following manually written
> rules?

This idea led to Artificial Intelligence.

------------------------------------------------------------------------

# Artificial Intelligence (AI)

Artificial Intelligence is the field of computer science that enables
machines to perform tasks that normally require human intelligence.

Examples:

-   ChatGPT
-   Google Translate
-   Google Maps
-   Tesla Self Driving
-   Face Recognition

------------------------------------------------------------------------

# Machine Learning (ML)

Machine Learning is a subset of AI.

Instead of manually writing rules, we provide data (examples).

The computer learns patterns from the data and makes predictions.

Workflow:

``` text
Training Data
      │
      ▼
Machine Learning Model
      │
      ▼
Pattern Learning
      │
      ▼
Prediction
```

Example:

-   5000 Apple images
-   5000 Banana images

The model learns the difference automatically.

------------------------------------------------------------------------

# Why Deep Learning?

Traditional Machine Learning often requires manual feature engineering.

For complex data like:

-   Images
-   Videos
-   Speech
-   Medical scans

manual feature extraction becomes difficult.

Deep Learning solves this problem.

------------------------------------------------------------------------

# Deep Learning (DL)

Deep Learning is a subset of Machine Learning that uses Artificial
Neural Networks with multiple hidden layers to automatically learn
features and patterns from data.

Traditional ML:

``` text
Image
 ↓
Human creates Features
 ↓
ML Model
 ↓
Prediction
```

Deep Learning:

``` text
Image
 ↓
Deep Neural Network
 ↓
Automatic Feature Learning
 ↓
Prediction
```

------------------------------------------------------------------------

# AI vs ML vs DL

  AI                        ML                   DL
  ------------------------- -------------------- -------------------------
  Broad field               Subset of AI         Subset of ML
  Human-like intelligence   Learns from data     Learns complex patterns
  Can use rules             Uses training data   Uses Neural Networks

# Summary

complex features using deep neural networks.
