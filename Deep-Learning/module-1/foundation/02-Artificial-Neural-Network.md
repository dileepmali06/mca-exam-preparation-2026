# Why Was ANN Created?

Early AI systems depended on manually written rules.

Example:

``` text
IF object has 4 legs
AND has a tail
AND barks
THEN Dog
```

Writing thousands of rules for every real-world situation became
impossible.

Researchers asked:

> "Can a computer learn patterns the same way humans learn?"

This idea led to Artificial Neural Networks.

------------------------------------------------------------------------

# Inspiration from the Human Brain

The human brain contains billions of biological neurons.

Each neuron:

-   Receives signals
-   Processes information
-   Sends signals to other neurons

Researchers created a mathematical version of this idea.

This mathematical model is called an **Artificial Neuron**.

# What is an Artificial Neuron?

An Artificial Neuron is the smallest processing unit of a Neural
Network.

Its job is to:

-   Receive inputs
-   Perform mathematical calculations
-   Produce an output

A single neuron is not very intelligent.

Real intelligence comes when thousands or millions of neurons are
connected together.

------------------------------------------------------------------------

# What is an Artificial Neural Network (ANN)?

An Artificial Neural Network (ANN) is a network of interconnected
artificial neurons that learns patterns from data and makes predictions.

ANN is the core technology behind Deep Learning.

------------------------------------------------------------------------

# Basic Architecture of ANN

``` text
Input Layer
      │
      ▼
Hidden Layer(s)
      │
      ▼
Output Layer
```

------------------------------------------------------------------------

# Input Layer

The Input Layer receives raw data.

Examples:

-   Image
-   Text
-   Audio
-   Sensor Data

It simply forwards the data to the next layer.

------------------------------------------------------------------------

# Hidden Layer

The Hidden Layer performs all learning and calculations.

This is where:

-   Pattern Recognition
-   Feature Learning
-   Decision Making

takes place.

Deep Learning models usually contain multiple hidden layers.

------------------------------------------------------------------------

# Output Layer

The Output Layer produces the final prediction.

Example:

``` text
Input Image
      │
      ▼
Neural Network
      │
      ▼
Cat (98%)
Dog (2%)
```

------------------------------------------------------------------------

# Why Are Neural Networks Powerful?

One neuron has limited capability.

Thousands or millions of neurons working together can learn highly
complex patterns.

This makes Neural Networks suitable for:

-   Face Recognition
-   Speech Recognition
-   Medical Diagnosis
-   Self Driving Cars
-   Large Language Models (ChatGPT)

# Summary

Artificial Neural Networks are mathematical models inspired by the human
brain.

They consist of interconnected artificial neurons arranged into input,
hidden and output layers.

ANNs are the foundation of modern Deep Learning systems and power
applications such as ChatGPT, self-driving cars, medical diagnosis and
image recognition.
