# Softmax Function in PyTorch

PyTorch provides a built-in Softmax function.

``` python
import torch.nn.functional as F

output = F.softmax(input_tensor, dim=1)
```

This converts raw scores (logits) into probabilities.

------------------------------------------------------------------------

# Understanding the Syntax

``` python
import torch.nn.functional as F
```

Imports the functional module.

------------------------------------------------------------------------

``` python
F.softmax()
```

Calls the Softmax function.

------------------------------------------------------------------------

``` python
input_tensor
```

Represents the tensor containing raw scores.

------------------------------------------------------------------------

``` python
dim
```

Specifies the dimension along which Softmax is applied.

------------------------------------------------------------------------

# Understanding the dim Parameter

Consider the following matrix:

``` text
1 2 3
4 5 6
7 8 9
```

## dim = 0

Softmax is applied column-wise.

## dim = 1

Softmax is applied row-wise.

------------------------------------------------------------------------

# Practical Example

``` python
import torch
import torch.nn.functional as F

scores = torch.tensor([2.0, 1.0, 0.5])

probabilities = F.softmax(scores, dim=0)

print(probabilities)
```

Possible Output

``` text
tensor([0.63, 0.23, 0.14])
```

Total Probability

``` text
0.63 + 0.23 + 0.14 = 1.00
```
