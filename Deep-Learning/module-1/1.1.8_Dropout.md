# Why Do We Need Dropout?

Sometimes a neural network memorizes the training data instead of
learning general patterns.

This causes excellent training accuracy but poor performance on unseen
data.

This problem is called **Overfitting**.

Dropout is used to reduce Overfitting.

------------------------------------------------------------------------

# Overfitting

## Definition

Overfitting occurs when a model memorizes the training data but performs
poorly on new data.

### Characteristics

-   High Training Accuracy
-   Low Testing Accuracy

------------------------------------------------------------------------

# Underfitting

## Definition

Underfitting occurs when the model cannot learn the training data
properly.

### Characteristics

-   Low Training Accuracy
-   Low Testing Accuracy

------------------------------------------------------------------------

# Comparison

  Overfitting               Underfitting
  ------------------------- -----------------------
  Memorizes training data   Learns too little
  High training accuracy    Low training accuracy
  Poor testing accuracy     Poor testing accuracy
  Complex model             Oversimplified model

------------------------------------------------------------------------

# Generalization

Generalization is the ability of a trained model to perform well on
unseen data.

A good Deep Learning model should generalize well.

------------------------------------------------------------------------

# What is Dropout?

Dropout is a **regularization technique** that randomly disables some
neurons during training.

These neurons are temporarily ignored so that the network learns more
robust features.

------------------------------------------------------------------------

# Working of Dropout

Without Dropout

``` text
Input
  │
  ▼
● ● ● ● ●
  │
  ▼
Output
```

With Dropout

``` text
Input
  │
  ▼
● X ● X ●
  │
  ▼
Output
```

X = Temporarily disabled neuron.

------------------------------------------------------------------------

# Training vs Testing

## During Training

-   Dropout is ON.
-   Random neurons are disabled.

## During Testing

-   Dropout is OFF.
-   All neurons remain active.

------------------------------------------------------------------------

# Dropout Rate

The dropout rate determines the probability of dropping a neuron.

Examples:

  Dropout Rate   Meaning
  -------------- ---------------------
  0.2            20% neurons dropped
  0.3            30% neurons dropped
  0.5            50% neurons dropped

------------------------------------------------------------------------

# PyTorch Implementation

``` python
import torch.nn as nn

dropout = nn.Dropout(p=0.5)
```

Here,

-   p = Dropout probability

------------------------------------------------------------------------

# Advantages

-   Reduces Overfitting.
-   Improves Generalization.
-   Prevents neuron co-dependency.
-   Easy to implement.
-   Improves robustness.

------------------------------------------------------------------------

# Limitations

-   Training becomes slightly slower.
-   Very high dropout may cause Underfitting.
-   Not required for every model.

# Practice MCQs

## Easy

### Q1. What problem does Dropout mainly solve?

A. Underfitting

B. Overfitting

C. Data Cleaning

D. Gradient Explosion

**Answer:** B

**Explanation:** Dropout reduces overfitting by randomly disabling
neurons.

------------------------------------------------------------------------

### Q2. During training, Dropout:

A. Keeps all neurons active

B. Randomly disables neurons

C. Deletes the dataset

D. Stops Backpropagation

**Answer:** B

**Explanation:** Some neurons are temporarily disabled during training.

------------------------------------------------------------------------

### Q3. During testing, Dropout is:

A. ON

B. OFF

C. Random

D. Unknown

**Answer:** B

**Explanation:** All neurons remain active during testing.

------------------------------------------------------------------------

### Q4. A Dropout rate of 0.5 means:

A. 5% neurons dropped

B. 50% neurons may be dropped

C. 100% neurons dropped

D. No neurons dropped

**Answer:** B

**Explanation:** Each neuron has a 50% chance of being dropped.

training.

# Final Summary

Dropout is a powerful regularization technique that reduces overfitting
by randomly disabling neurons during training. This forces the network
to learn robust and independent features, improving generalization and
performance on unseen data.
