## Training

Training is the process of teaching a neural network using data.

## Forward Propagation

Input passes through hidden layers to generate a prediction.

## Prediction

The output generated by the neural network.

## Ground Truth

The actual correct answer used for comparison.

## Error

Difference between prediction and actual answer.

## Loss Function

Measures how wrong the prediction is. Goal: Minimize the loss.

### MSE

Used mainly for regression problems.

### Cross Entropy Loss

Used mainly for classification problems and commonly with Softmax.

## Backpropagation

Uses the calculated loss to update the weights.

## Gradient

Shows the direction in which the loss changes.

## Gradient Descent

Optimization algorithm used to reduce loss.

## Learning Rate

Controls how much weights change during each update.

## Weight Update

Weights are adjusted to improve predictions.

## Batch

A small subset of the training dataset processed together.

## Epoch

One complete pass through the entire training dataset.

# Summary

Neural Network training begins with Forward Propagation to generate
predictions. The prediction is compared with the Ground Truth using a
Loss Function. Backpropagation computes how weights should change.
Gradient Descent updates the weights using the Learning Rate. This
process repeats over batches and epochs until the model performs well.
