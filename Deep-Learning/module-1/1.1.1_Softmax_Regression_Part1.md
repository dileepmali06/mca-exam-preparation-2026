# Why Do We Need Softmax Regression?

Many real-world AI problems require selecting one class from multiple
possible classes.

Examples:

-   Image Classification
-   Traffic Sign Recognition
-   Fashion Item Classification
-   Animal Recognition
-   Disease Classification

Traditional Logistic Regression works well only for **two classes**.

Researchers therefore developed **Softmax Regression** to solve
multi-class problems.

------------------------------------------------------------------------

# Binary Classification

Binary Classification means there are only **two possible classes**.

Examples:

-   Spam / Not Spam
-   Cat / Dog
-   Healthy / Diseased
-   Pass / Fail

Example Diagram

``` text
Input
  │
  ▼

  │
  ├── Cat
  └── Dog
```

# Multi-Class Classification

Examples:

# The Problem with Logistic Regression

Logistic Regression is mainly designed for binary classification.

Example:

``` text
Cat
or
Dog
```

However, it cannot directly predict among many mutually exclusive
classes such as:

``` text
Cat
Dog
Horse
Cow
Elephant
Lion
```

Researchers therefore extended Logistic Regression.

The new technique became **Softmax Regression**.

------------------------------------------------------------------------

# What is Softmax Regression?

Softmax Regression is the multi-class extension of Logistic Regression.

It converts the output of a neural network into probabilities for
multiple classes.

The class with the highest probability becomes the final prediction.

------------------------------------------------------------------------

# Example

Input Image

↓

Fruit Image

↓

Softmax Output

  Class      Probability
  -------- -------------
  Apple              95%
  Banana              2%
  Orange              1%
  Mango               1%
  Grapes              1%

Prediction:

**Apple**

------------------------------------------------------------------------

# What is Probability?

Probability represents the likelihood that an event will happen.

Range:

``` text
0  -------------------------> 1

Impossible              Certain
```

Examples:

-   0 → Impossible
-   0.50 → 50% Chance
-   1 → Guaranteed

------------------------------------------------------------------------

# Special Property of Softmax

All output probabilities always add up to **1 (100%)**.

Example:

  Class      Probability
  -------- -------------
  Apple             0.70
  Banana            0.10
  Orange            0.08
  Mango             0.07
  Grapes            0.05

Total:

``` text
0.70
+0.10
+0.08
+0.07
+0.05
------
1.00
```

This is called a **Probability Distribution**.

# Summary

Softmax Regression is a generalized version of Logistic Regression
designed for Multi-Class Classification problems.

Instead of predicting only two classes, it calculates the probability of
every possible class and selects the class with the highest probability
as the final prediction.

Softmax is widely used in the output layer of Deep Learning models for
image classification, medical diagnosis, recommendation systems,
autonomous vehicles and many other AI applications.
