# Why Do We Need Activation Functions?

A neural network without activation functions behaves like a linear
model, no matter how many hidden layers it has.

Activation Functions introduce **non-linearity**, allowing neural
networks to learn complex real-world patterns.

Examples:

-   Image Recognition
-   Speech Recognition
-   Medical Diagnosis
-   Natural Language Processing
-   Self-Driving Cars

------------------------------------------------------------------------

# What is an Activation Function?

An Activation Function is a mathematical function that decides whether a
neuron should be activated and what output should be passed to the next
layer.

Simple Flow:

``` text
Input
   │
   ▼
Weighted Sum
   │
   ▼
Activation Function
   │
   ▼
Output
```

# Linear vs Non-Linear

## Linear

-   Straight-line relationship.
-   Output changes proportionally with input.
-   Cannot model complex real-world data.

## Non-Linear

-   Complex relationships.
-   Learns patterns from images, text, audio and video.
-   Makes Deep Learning possible.

# Types of Activation Functions

## 1. Linear Activation

### Definition

Output is directly proportional to input.

### Output Range

(-∞, +∞)

### Advantages

-   Simple
-   Useful for regression output layer

### Disadvantages

-   Cannot learn complex patterns.
-   Not suitable for hidden layers.

### Applications

-   House Price Prediction
-   Regression Output Layer

------------------------------------------------------------------------

## 2. Sigmoid Activation

### Definition

Maps input values between **0 and 1**.

### Output Range

0 to 1

### Advantages

-   Produces probability-like output.
-   Suitable for binary classification.

### Disadvantages

-   Vanishing Gradient Problem.
-   Slow convergence in deep networks.

### Applications

-   Spam Detection
-   Fraud Detection
-   Binary Disease Prediction

------------------------------------------------------------------------

## 3. Tanh Activation

### Definition

Maps values between **-1 and 1**.

### Output Range

-1 to 1

### Advantages

-   Zero-centered output.
-   Better than Sigmoid in many cases.

### Disadvantages

-   Still suffers from Vanishing Gradient.

### Applications

-   Traditional RNN models.

------------------------------------------------------------------------

## 4. ReLU (Rectified Linear Unit)

### Rule

If x \< 0 → Output = 0

If x \> 0 → Output = x

### Output Range

0 to +∞

### Advantages

-   Fast
-   Simple
-   Reduces Vanishing Gradient
-   Most popular hidden layer activation

### Disadvantages

-   Dead Neuron Problem

### Applications

-   CNN
-   Computer Vision
-   Image Classification

------------------------------------------------------------------------

## 5. Leaky ReLU

### Purpose

Solves the Dead Neuron Problem of ReLU.

### Rule

Positive input → Same output

Negative input → Small negative output

### Advantages

-   Prevents dead neurons.
-   Better gradient flow.

### Applications

-   Deep CNNs
-   Object Detection

------------------------------------------------------------------------

## 6. Softmax

### Definition

Converts raw scores (logits) into a probability distribution.

### Output

All probabilities sum to 1.

### Applications

-   Multi-Class Classification
-   Image Classification
-   Language Models

------------------------------------------------------------------------

# Comparison Table

  Activation   Output Range               Best Use
  ------------ -------------------------- ----------------------------
  Linear       (-∞, +∞)                   Regression Output
  Sigmoid      0 to 1                     Binary Classification
  Tanh         -1 to 1                    Traditional RNN
  ReLU         0 to +∞                    Hidden Layers
  Leaky ReLU   Small negative to +∞       Deep Networks
  Softmax      Probability Distribution   Multi-Class Classification

# One-Line Definitions

-   **Activation Function:** Decides the output of a neuron.
-   **Sigmoid:** Maps values between 0 and 1.
-   **Tanh:** Maps values between -1 and 1.
-   **ReLU:** Returns 0 for negative inputs and input itself for
    positive values.
-   **Leaky ReLU:** Improved ReLU with a small negative slope.
-   **Softmax:** Converts logits into probabilities.

# Practice MCQs

## Easy

### Q1. What is the primary purpose of an Activation Function?

A. Store data

B. Add non-linearity

C. Increase dataset size

D. Reduce epochs

**Answer:** B

**Explanation:** Activation Functions introduce non-linearity into
neural networks.

### Q2. Which activation is mainly used for binary classification?

A. ReLU

B. Sigmoid

C. Softmax

D. Linear

**Answer:** B

**Explanation:** Sigmoid outputs values between 0 and 1.

### Q3. Which activation converts logits into probabilities?

A. ReLU

B. Tanh

C. Softmax

D. Linear

**Answer:** C

**Explanation:** Softmax creates a probability distribution.

A. Softmax

B. ReLU

D. Sigmoid

**Explanation:** ReLU is fast and effective for hidden layers.

# Final Summary

Activation Functions are one of the most important components of Deep
Learning. They introduce non-linearity, allowing neural networks to
solve complex real-world problems. Among the major activation functions,
ReLU is commonly used in hidden layers, Sigmoid is suitable for binary
classification, Softmax is used for multi-class classification, Tanh
provides zero-centered outputs, and Linear activation is generally used
in regression output layers.
