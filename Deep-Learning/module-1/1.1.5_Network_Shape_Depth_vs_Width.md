# Why Do We Need Network Shape?

The architecture of a neural network affects:

-   Learning capability
-   Model complexity
-   Training time
-   Accuracy
-   Memory usage

Choosing the correct network shape is an important design decision.

------------------------------------------------------------------------

# What is Depth?

**Depth** is the number of hidden layers between the input layer and
output layer.

Example:

``` text
Input
  │
  ▼
Hidden Layer 1
  │
  ▼
Hidden Layer 2
  │
  ▼
Hidden Layer 3
  │
  ▼
Output
```

Depth = 3 Hidden Layers

------------------------------------------------------------------------

# What is Width?

**Width** is the number of neurons present in a hidden layer.

Example:

Narrow Layer

``` text
● ● ●
```

Wide Layer

``` text
● ● ● ● ● ● ● ● ● ●
```

More neurons = More Width.

------------------------------------------------------------------------

# Deep Network

Characteristics:

-   More hidden layers
-   Learns hierarchical features
-   Suitable for complex problems
-   Used in Deep Learning

------------------------------------------------------------------------

# Wide Network

Characteristics:

-   More neurons per layer
-   Simpler architecture
-   Suitable for relatively simple problems

------------------------------------------------------------------------

# Deep vs Wide Network

  Deep Network                    Wide Network
  ------------------------------- --------------------------------
  More hidden layers              More neurons per layer
  Better feature hierarchy        More parallel feature learning
  Higher computational cost       Lower depth
  Used in CNNs and Transformers   Used in simpler neural models

------------------------------------------------------------------------

# Why is Deep Learning Called "Deep"?

Deep Learning uses multiple hidden layers.

A neural network with many hidden layers is called a **Deep Neural
Network (DNN)**.

# Summary

Network Shape describes the architecture of a neural network. **Depth**
represents the number of hidden layers, while **Width** represents the
number of neurons in each hidden layer. Deep Networks are suitable for
solving complex AI problems, whereas Wide Networks increase the
processing capacity within a single layer. Understanding these concepts
is essential before studying advanced architectures such as CNNs, RNNs,
and Transformers.
