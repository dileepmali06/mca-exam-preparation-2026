# Why Do We Need the Softmax Function?

A Neural Network does not directly produce probabilities.

Instead, it produces **raw scores**, called **logits**.

Example:

``` text
Cat   = 8.5
Dog   = 5.2
Horse = 2.1
```

These values are difficult to interpret.

Softmax converts these raw scores into probabilities.

------------------------------------------------------------------------

# What are Logits?

Logits are the raw output values generated by the final layer of a
neural network before applying Softmax.

Example:

``` text
Cat   = 8.5
Dog   = 5.2
Horse = 2.1
```

These are **not probabilities**.

------------------------------------------------------------------------

# After Applying Softmax

``` text
Cat   = 95%
Dog   = 4%
Horse = 1%
```

Now the prediction is easy to understand.

------------------------------------------------------------------------

# Real-Life Analogy

Suppose three students scored:

# Softmax Workflow

``` text
Raw Scores (Logits)
        │
        ▼
Exponentiation
        │
        ▼
Normalization
        │
        ▼
Probability Distribution
        │
        ▼
Prediction
```

------------------------------------------------------------------------

# Intuition Behind the Formula

The Softmax equation ensures:

-   Higher score → Higher probability
-   Lower score → Lower probability
-   No negative probabilities
-   Total probability always equals 1

# Softmax vs Sigmoid

  Sigmoid                  Softmax
  ------------------------ ----------------------------
  Binary Classification    Multi-Class Classification
  Two classes              Three or more classes
  Independent outputs      Probability distribution
  Output between 0 and 1   All outputs sum to 1

------------------------------------------------------------------------

# Advantages

-   Converts logits into probabilities.
-   Supports Multi-Class Classification.
-   Easy to interpret.
-   Used in modern Deep Learning models.
-   Works well with Cross-Entropy Loss.

------------------------------------------------------------------------

# Limitations

-   Not required for simple binary problems.
-   Computational cost increases with many classes.
-   Very confident predictions are not always correct.

# Summary

The Softmax Function converts the raw output scores (logits) of a neural
network into a valid probability distribution.

It ensures that all probabilities lie between 0 and 1 and that their
total equals 1. The class with the highest probability is selected as
the final prediction.

Softmax is one of the most important functions used in the output layer
of Deep Learning models for multi-class classification.
