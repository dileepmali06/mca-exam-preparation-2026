# Why Do We Need GPU?

Deep Learning models perform millions of mathematical operations.

Using only a CPU can make training very slow.

A GPU performs many calculations simultaneously, making training much
faster.

------------------------------------------------------------------------

# What is CPU?

**Definition**

CPU (Central Processing Unit) is the main processor of a computer that
performs general-purpose tasks.

Examples:

-   Running the operating system
-   Web browsing
-   Programming
-   Office applications

------------------------------------------------------------------------

# What is GPU?

**Definition**

GPU (Graphics Processing Unit) is a processor designed for highly
parallel computations.

It can execute thousands of operations simultaneously, making it ideal
for Deep Learning.

------------------------------------------------------------------------

# CPU vs GPU

  CPU                         GPU
  --------------------------- ----------------------------------------
  General-purpose processor   Parallel processor
  Few powerful cores          Hundreds or thousands of smaller cores
  Sequential processing       Parallel processing
  Slower for Deep Learning    Faster for Deep Learning

------------------------------------------------------------------------

# Why Deep Learning Uses GPUs

Neural networks repeatedly perform:

-   Matrix Multiplication
-   Matrix Addition
-   Activation Functions
-   Gradient Calculations

GPUs execute these operations in parallel, greatly reducing training
time.

------------------------------------------------------------------------

# Parallel Processing

``` text
CPU

Task 1
 ↓
Task 2
 ↓
Task 3

-----------------------

GPU

Task 1   Task 2   Task 3

Executed Simultaneously
```

------------------------------------------------------------------------

# What is CUDA?

**CUDA (Compute Unified Device Architecture)** is NVIDIA's platform that
allows software such as PyTorch and TensorFlow to use NVIDIA GPUs.

Simple Meaning:

CUDA is the bridge between Deep Learning software and NVIDIA GPUs.

------------------------------------------------------------------------

# GPU Training Workflow

``` text
Dataset
   │
   ▼
CPU Loads Data
   │
   ▼
GPU Receives Data
   │
   ▼
Forward Propagation
   │
   ▼
Loss Calculation
   │
   ▼
Backpropagation
   │
   ▼
Weight Update
   │
   ▼
Repeat
```
