# Why Was Convolution Invented?

Traditional Artificial Neural Networks (ANNs) work well for numerical
data but struggle with images.

Problems with ANN for images:

-   Huge number of input neurons.
-   Very high memory consumption.
-   Slow training.
-   Loss of spatial information.
-   High risk of overfitting.

Researchers solved this by introducing **Convolution**, which extracts
useful features from small regions of an image.

------------------------------------------------------------------------

# What is Convolution?

**Definition**

Convolution is a mathematical operation used to extract important
features (edges, corners, textures, patterns) from an image.

Simple Meaning:

Instead of looking at the whole image at once, the model scans small
regions and learns useful visual patterns.

------------------------------------------------------------------------

# Image Representation

Humans see:

🐶 Dog

Computer sees:

``` text
125 130 140
200 180 160
90 100 110
```

These numbers represent **pixel values**.

------------------------------------------------------------------------

# What is a Pixel?

A Pixel (Picture Element) is the smallest unit of a digital image.

Example:

``` text
🟥 🟩 🟦
```

Each colored square represents one pixel.

------------------------------------------------------------------------

# Kernel (Filter)

A **Kernel** (also called a **Filter**) is a small matrix that slides
over the image.

Example:

``` text
1  0 -1
1  0 -1
1  0 -1
```

Different kernels detect different features.

------------------------------------------------------------------------

# Sliding Window

The kernel moves across the image one region at a time.

``` text
Image
+----+----+
|1 2 |2 3 |
|5 6 |6 7 |
+----+----+
```

This movement is called the **Sliding Window**.

------------------------------------------------------------------------

# Feature Extraction

The kernel extracts useful features such as:

-   Edges
-   Corners
-   Texture
-   Shapes
-   Patterns

------------------------------------------------------------------------

# Feature Map

The output generated after applying convolution is called the **Feature
Map** (or Activation Map).

Flow:

``` text
Image
   │
   ▼
Kernel
   │
   ▼
Convolution
   │
   ▼
Feature Map
```

------------------------------------------------------------------------

# Complete Working

``` text
Input Image
      │
      ▼
Kernel / Filter
      │
      ▼
Sliding Window
      │
      ▼
Multiply & Add
      │
      ▼
Feature Map
      │
      ▼
CNN
```

------------------------------------------------------------------------

# Why CNN Uses Convolution

-   Reduces parameters.
-   Preserves spatial information.
-   Learns important visual features.
-   Improves image recognition accuracy.

------------------------------------------------------------------------

# PyTorch Example

``` python
import torch.nn as nn

conv = nn.Conv2d(
    in_channels=3,
    out_channels=32,
    kernel_size=3
)
```

Meaning:

-   3 input channels (RGB)
-   32 filters
-   3×3 kernel size

# Final Summary

Convolution is the foundation of Convolutional Neural Networks (CNNs).
It allows a computer to analyze images by scanning them with small
kernels that extract meaningful features such as edges, corners and
textures. The output of this operation is called a Feature Map, which is
further processed by CNN layers for image recognition and
classification.
