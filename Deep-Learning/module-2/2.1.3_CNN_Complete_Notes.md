# Why Was CNN Invented?

Artificial Neural Networks (ANNs) perform well on numerical data but
struggle with images because:

-   Huge number of input pixels
-   High memory usage
-   Slow training
-   Poor image feature extraction
-   Overfitting

CNN solves these problems by automatically extracting image features.

------------------------------------------------------------------------

# What is CNN?

**Definition**

A Convolutional Neural Network (CNN) is a Deep Learning model specially
designed to process images and other grid-like data.

Simple Meaning:

CNN learns important visual features (edges, shapes, textures) and uses
them to classify images.

# Layers of CNN

## 1. Input Layer

Receives the image.

Example:

224 × 224 × 3 RGB image.

------------------------------------------------------------------------

## 2. Convolution Layer

Extracts features such as:

-   Edges
-   Corners
-   Shapes
-   Textures

------------------------------------------------------------------------

## 3. ReLU Layer

-   Removes negative values.
-   Introduces non-linearity.
-   Speeds up learning.

------------------------------------------------------------------------

## 4. Max Pooling Layer

-   Reduces feature map size.
-   Preserves important information.
-   Makes computation faster.

------------------------------------------------------------------------

## 5. Flatten Layer

Converts a 2D feature map into a 1D vector.

Example:

``` text
1 2
3 4

↓

1 2 3 4
```

------------------------------------------------------------------------

## 6. Fully Connected Layer

Combines all extracted features and performs classification.

------------------------------------------------------------------------

## 7. Softmax Layer

Converts output scores into probabilities.

Example:

Dog = 98%

Cat = 1%

Horse = 1%

# CNN vs ANN

  ANN                          CNN
  ---------------------------- ------------------------------
  General-purpose network      Image-focused network
  Manual feature learning      Automatic feature extraction
  Large number of parameters   Fewer parameters
  Less effective for images    Highly effective for images

# PyTorch Example

``` python
import torch.nn as nn

model = nn.Sequential(
    nn.Conv2d(3, 32, 3),
    nn.ReLU(),
    nn.MaxPool2d(2),
    nn.Flatten(),
    nn.Linear(5408, 10)
)
```

# Practice MCQs

## Easy

### Q1. CNN is mainly used for:

A. Database Management

B. Image Processing

C. Networking

D. Compiler Design

**Answer:** B

**Explanation:** CNN is specially designed for image-related tasks.

------------------------------------------------------------------------

### Q2. Which layer extracts image features?

A. Flatten

B. Convolution

C. Softmax

D. Fully Connected

**Answer:** B

**Explanation:** Convolution extracts edges, textures and shapes.

------------------------------------------------------------------------

### Q3. Which layer reduces feature map size?

A. Pooling

B. Flatten

C. Input

D. Softmax

**Answer:** A

**Explanation:** Pooling downsamples the feature map.

------------------------------------------------------------------------

### Q4. Flatten converts:

A. 1D → 2D

B. 2D → 1D

C. RGB → Gray

D. Image → Text

**Answer:** B

**Explanation:** Flatten converts matrices into vectors.
