# What is a Pre-trained Network?

A **Pre-trained Network** is a Deep Learning model that has already been
trained on a very large dataset (such as **ImageNet**) and can be reused
for new tasks.

**Simple Meaning**

Instead of training a model from scratch, we reuse an already trained
model and adapt it for our own problem.

------------------------------------------------------------------------

# Why Use a Pre-trained Network?

Without a pre-trained model:

-   Large dataset required
-   Long training time
-   High computational cost

With a pre-trained model:

-   Small dataset is often enough
-   Faster training
-   Better starting accuracy
-   Lower cost

------------------------------------------------------------------------

# Transfer Learning

**Definition**

Transfer Learning is the process of reusing knowledge learned from one
task to solve another related task.

Example:

``` text
ImageNet Model
      │
      ▼
Already Learned Features
      │
      ▼
New Dataset
      │
      ▼
New Prediction
```

------------------------------------------------------------------------

# Fine-Tuning

Fine-Tuning means retraining some or all layers of a pre-trained model
to improve performance on a new dataset.

Benefits:

-   Better accuracy
-   Better adaptation
-   Less training than starting from scratch

------------------------------------------------------------------------

# Practice MCQs

## Easy

### Q1. A Pre-trained Network is:

A. A new model

B. An already trained model

C. A database

D. A GPU

**Answer:** B

**Explanation:** It has already learned from a large dataset.

------------------------------------------------------------------------

### Q2. Transfer Learning means:

A. Reusing learned knowledge

B. Deleting layers

C. Increasing epochs

D. Compressing images

**Answer:** A

**Explanation:** Existing knowledge is transferred to a related task.

------------------------------------------------------------------------

### Q3. Which dataset is commonly used for pre-training?

A. Iris

B. CIFAR-10

C. ImageNet

D. MNIST

**Answer:** C

**Explanation:** ImageNet is the most widely used dataset for
pre-training CNN models.

------------------------------------------------------------------------

### Q4. Which model is optimized for mobile devices?

A. ResNet

B. MobileNet

C. VGG

D. AlexNet

**Answer:** B

**Explanation:** MobileNet is lightweight and efficient.

------------------------------------------------------------------------

### Q5. Fine-Tuning means:

A. Further training a pre-trained model

B. Deleting the model

C. Reducing image size

D. Increasing image resolution

**Answer:** A

**Explanation:** Fine-Tuning adapts a pre-trained model to a new task.

------------------------------------------------------------------------

# Final Summary

A Pre-trained Network is an already trained Deep Learning model that can
be reused for new tasks. By using **Transfer Learning** and
**Fine-Tuning**, developers can build accurate AI systems with less
data, less computation and much faster training than training a model
from scratch.
